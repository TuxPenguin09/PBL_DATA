# A C# Winforms Project made by iACADEMY students
### Topic: Gender Equality
SameScope is a data visualizing application for the topic and it can also see how much of the country, people, etc. are in favor, support and increased leadership analytics.
## For groupmates that are known to programming
Please create your own branch for adding something in the app, after that, you may create a pull request with a base of `master` and name it the feature you wish to add for the app, I'll check it out and fixing after.
